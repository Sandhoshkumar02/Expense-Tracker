## ⚙️ Setup Instructions

### Clone Repo

```bash
git clone https://github.com/Sandhoshkumar02/Expense-Tracker.git
```

### Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
